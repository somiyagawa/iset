# ISET 2026 — The 2nd International Symposium of Egyptology in Tokyo

第2回 東京エジプト学国際シンポジウム（ISET 2026）

A single-day symposium held **the next day after [CRE26 Tokyo](https://cre26.tokyo/)** — the Current Research in Egyptology conference.

- **Date:** 16 July 2026 (Thursday)
- **Venue:** University of Tsukuba, Tokyo Campus, Lecture Room 120 (Myōgadani, Bunkyō, Tokyo) — also streamed online
- **Registration:** https://forms.gle/CTEYaVueAddmYmMF7

## Contents

- `index.html` — the symposium website (single file, no build step)
- `images/` — venue and city photographs

## Viewing

Open `index.html` in any modern browser.

## Hosting on GitHub Pages

After pushing this repository, enable **Settings → Pages → Source: `main` / root**.
The site will be served at `https://<your-username>.github.io/iset2026/`.
